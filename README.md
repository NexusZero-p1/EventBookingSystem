# EventBookingSystem
Система бронирования билетов на мероприятия 

## Установка
1) Поставить Visual Studio и SQL Server
2) Запустить скрипт базы данных из папки Database
3) Открыть проект в Visual Studio
4) Настроить подключение к базе в Form1.cs
5) Запустить программу

## Функции
1) Просмотр мероприятий
2) Поиск по дате
3) Фильтры по жанру
4) Бронирование билетов

Разработано на C# + SQL Server
